# redmine_book
A util python script to book time entry on redmine.

Usages:
    
    $ python book.py

You can also create a task on Windows: 

    > schtasks /Create /?

Notes: Before execute *book.py*, please reset the corresponding setting item in the *settings.py*.

+ REDMINE_URL
+ REDMINE_KEY
+ ISSUE_ID
+ HOURS
+ ACTIVITY_ID
+ COMMENTS
